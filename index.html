Parcours

Réseaux

Objectif : Acquérir une compréhension approfondie des concepts de base et avancés des réseaux informatiques.
Description : Cette section couvre les fondamentaux des réseaux, y compris les différences entre le Wi-Fi et les réseaux filaires, les types de stockage de données, et les protocoles de communication essentiels.
Compétences acquises :

  • Différenciation entre Wi-Fi et réseaux filaires : Comprendre les avantages et les inconvénients de chaque type de connexion, ainsi que leurs cas d'utilisation.
  • Types de stockage : Exploration des solutions de stockage telles que les disques durs (HDD), les disques SSD, et le stockage en cloud.
  • Protocoles de messagerie : Maîtrise des protocoles POP, IMAP et SMTP pour la gestion des emails.
  • Mission 2ème année M2L : Participation à des projets concrets pour la mise en œuvre de serveurs, la configuration de la commutation et du routage, ainsi que la mise en place de relais DHCP.

Spécialité

Objectif : Approfondir les connaissances techniques dans des domaines spécialisés des réseaux et des systèmes.
Description : Cette section se concentre sur des concepts avancés tels que l'agrégation de liens, IPv6, le routage dynamique, et les solutions de haute disponibilité.
Compétences acquises :

  • Agrégation de liens : Combinaison de plusieurs connexions réseau pour augmenter la bande passante et la redondance.
  • IPv6 : Comprendre et configurer le protocole IPv6 pour répondre aux besoins croissants en adresses IP.
  • Routage dynamique : Utilisation de protocoles comme OSPF et BGP pour optimiser le trafic réseau.
  • Spanning Tree : Prévention des boucles réseau dans les topologies complexes.
  • Haute disponibilité : Mise en place de solutions pour assurer la continuité des services.
  • NAT, PAT, NAS : Configuration des technologies de traduction d'adresses réseau et de stockage en réseau.

Sécurité informatique

Objectif : Protéger les données et les systèmes de la M2L contre les menaces externes et internes.
Description : Cette section aborde les aspects clés de la sécurité informatique, y compris la mise en place de politiques de sécurité, la configuration des règles de filtrage, et la sensibilisation des utilisateurs.
Compétences acquises :

  • Politiques de sécurité : Déploiement de pare-feu, antivirus, et anti-malware pour protéger les systèmes.
  • Règles de filtrage : Configuration des règles de contrôle d'accès pour limiter les accès non autorisés.
  • Sensibilisation des utilisateurs : Formation des utilisateurs aux bonnes pratiques de sécurité pour réduire les risques.
  • Audit de sécurité : Identification et correction des vulnérabilités pour renforcer la sécurité globale.

Mission 4 : Mise en place d'un DHCP relais et configuration avec Active Directory

Objectif : Simplifier la gestion des adresses IP sur le réseau de la M2L en déployant un serveur DHCP centralisé avec relais, intégré à Active Directory.
Tâches principales :

  • Installation du rôle DHCP : Ajout du rôle "Serveur DHCP" via le Gestionnaire de serveur.
  • Configuration des étendues DHCP : Définition des plages d'adresses IP, durée de bail, etc.
  • Activation du relais DHCP : Utilisation de l'option IP Helper sur les routeurs ou switchs compatibles.
  • Intégration avec Active Directory : Autorisation du serveur DHCP dans Active Directory pour éviter les conflits.
  • Tests et validation : Vérification que les clients réseau obtiennent bien une adresse IP via le serveur DHCP.
  • Documentation : Rédaction d'un guide de configuration pour le DHCP relais et son intégration avec Active Directory.
